মাঝে মাঝে ইউজার ইন্টারঅ্যাকশনের উপর ভিত্তি করে আমাদের ওয়েবসাইটের স্ক্রীন আপডেট করতে হয়। যেমন, গ্যালারির কোনো একটা ইমেজে ক্লিক করলে সেটা বড় করে দেখাবো বা ঐ ইমেজকে একটিভ হিসেবে দেখাবো।

**রিয়েক্টে সময়ের সাথে UI এর ডাটা পরিবর্তনকে state বলা হয়।** আমরা যেকোনো কম্পোনেন্টে `state` যোগ করতে পারি, এবং ইউজারের ইন্টারঅ্যাকশনের মাধ্যমে সেই state আপডেট করতে পারি।

আমরা এই পাঠে শিখবো:

- ইউজারের ইন্টারঅ্যাকশন হ্যান্ডল করতে, কীভাবে কম্পোনেন্টে ইভেন্ট হ্যান্ডলার ব্যবহার করবেন
- কিভাবে `state` ব্যবহার করে কম্পোনেন্টগুলো ডাটা মনে রাখে
- React কীভাবে UI আপডেট করে দুটি ধাপে
- কেন state পরিবর্তনের সাথে সাথে UI আপডেট হয় না
- কিভাবে একাধিক state আপডেট কিউয়ে থাকে এবং কাজ করে
- কিভাবে state এর অবজেক্ট আপডেট করবেন
- কিভাবে state এর অ্যারে আপডেট করবেন
