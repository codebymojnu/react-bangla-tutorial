ধরেন:

আপনি একটা রেস্টুরেন্টে আছেন এখন আপনি ওয়েটার কে ১ গ্লাস পানি দিতে বললেন।

এখানে দুইটা ঘটনা ঘটতে পারে --

১: ওয়েটার পানি এনে দিলো (✅ Success)
২: ওয়েটার পানি আনলো না (❌ Fail)

এটাই হলো Promise এর কাহিনী।
আমরা ওয়েটারের উপর ভরসা করছি, কিন্তু নিশ্চিত না -- যে পানি আসবে নাকি আসবে না।

চলুন কোড এক্সাম্পল দেখি:

```
// একটা প্রমিজ বানালাম পানি আনার জন্য

const paniPromise = new Promise(function(resolve, reject) {
  let waiterAvailable = true; // ওয়েটার আছে কিনা

// চেক করা হচ্ছে ওয়েটার আছে নাকি থাকলে পানি দিবে নাহলে দিবে না

if (waiterAvailable) {
      setTimeout(function() {
        resolve('ওয়েটার পানি এনে দিলো ২ সেকেন্ড পরে!');
      }, 2000);
 else {
    reject('ওয়েটার নাই, পানি আসলো না...');
  }
});
```

এখানে কী হচ্ছে?

১: প্রমিজ বানানো হলো
`new Promise(function(resolve, reject) {...})`

২: resolve = ওয়েটার পানি এনে দিলো (ওয়াদা রাখা হলো)

৩: reject = ওয়েটার পানি আনলো না (ওয়াদা ভেঙে গেলো)

এখন এই Promise-টা ব্যবহার করবো:

```
paniPromise
  .then(function(successMessage) {
    console.log(successMessage); // যদি পানি পাওয়া যায়
  })
  .catch(function(errorMessage) {
    console.log(errorMessage); // যদি পানি না পাওয়া যায়
  });
```

Step by Step Flow:

১: ওয়েটারকে পানি আনতে বললাম (Promise তৈরি হলো)

২: যদি ওয়েটার পানি আনে → `.then()` চলে

৩: যদি ওয়েটার পানি না আনে → `.catch()` চলে

আশা করি বুঝতে পেরেছেন ভুল হলে ক্ষমার দৃষ্টিতে দেখবেন ধন্যবাদ ✨
