### 👨‍💻 Role Overview

We are seeking an experienced Senior Front End Developer who lives and breathes JavaScript and React. You’ll be crafting smooth user interfaces, pushing performance boundaries, and animating delightful interactions.

### ✅ Responsibilities

- Develop high-quality front-end applications using React
- Implement performance optimizations and Server-Side Rendering (SSR)
- Create and maintain animated UI elements with GSAP, Framer Motion, and SVG animations
- Collaborate with UI/UX designers and backend developers
- Maintain code quality through testing and reviews
- Participate in team discussions to shape project direction

### 🔧 Requirements

- Extensive knowledge of core JavaScript (ES6+)
- Strong experience with React and ecosystem tools
- Solid understanding of React performance optimizations and SSR methods
- Proficiency with animation libraries like Framer Motion, GSAP, and SVG animations
- Good eye for UI/UX and performance
- Must be willing to work full-time on-site at our Tangail office  
  _(Remote work is not available for this role)_
