## tailwind.config.js setup

```tailwindcss
theme: {
 extend: {
   // Add custom spacing
   container: {
     center: true,
     padding: "2rem",
   },
   // Add custom colors
   colors: {
     primary: "#ff0000",
     secondary: "#00ff00",
   },
 },
},
```
