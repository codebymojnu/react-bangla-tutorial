The JavaScript `.some()` method is used to **check if at least one element** in an array satisfies a given condition.

---

### ‚úÖ Syntax:

```js
array.some(callback);
```

- `callback` is a function that runs for each element.
- If **any element** returns `true`, `.some()` returns `true`.
- If **none match**, it returns `false`.

---

### ‚úÖ Simple Example:

```js
const numbers = [1, 3, 5, 7, 8];

const hasEven = numbers.some((num) => num % 2 === 0);

console.log(hasEven);
// Output: true (because 8 is even)
```

---

### ‚úÖ Object Example:

```js
const users = [
  { id: 1, name: "Arafat" },
  { id: 2, name: "Hasan" },
];

const hasHasan = users.some((user) => user.name === "Hasan");

console.log(hasHasan);
// Output: true
```

---

### üìå Use Cases:

- Check if cart has any items.
- Check if a user is logged in.
- Check if any product is out of stock.

---

### ‚ö†Ô∏è Note:
