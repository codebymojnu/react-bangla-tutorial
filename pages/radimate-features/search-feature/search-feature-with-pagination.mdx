আমাদের চাওয়া,

১। এখানে আমাদের প্রথমত টার্গেট, ইউজার সবার শেষে যে ক্যারেকটার টাইপ করে থেমে যায়, তখন পুরো স্ট্রিং (মানে ইউজারের শেষে 'o' লেখার পর 'Hello' হয়) এর জন্য যে রেন্সপন্স আসবে তা UI এ দেখানো।

২। প্রতি কি স্ট্রোকে আমাদের API কল করার দরকার নাই। যখন ইউজার ৫০০ মিলিসেকন্ড থেমে যাবে, তখনই আমরা API কল করব। যেটাকে আমরা Debounce System বলি।
